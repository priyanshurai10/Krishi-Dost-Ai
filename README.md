# 🌾 Krishi Dost - AI-Powered Farming Assistant

Krishi Dost is a full-stack agritech platform built to empower Indian farmers. It provides real-time weather, market rates, and AI-driven agricultural intelligence.

## ✨ Features
- **Smart Pincode Telemetry:** Get live weather and district details instantly.
- **Multimodal AI Assistant:** Use Llama 3 Vision to identify crop diseases via photos.
- **Market Intelligence:** Live Mandi rates and agri-chemical database.
- **Financial Tools:** Built-in EMI and profit calculators.

## 💻 Tech Stack
- **Frontend:** React.js, Tailwind CSS, Vite
- **Backend:** Node.js, Express.js
- **Deployment:** Vercel & Render

## 🛠️ Run Locally
1. `git clone https://github.com/priyanshurai10/Krishi-Dost-Ai.git`
2. `cd backend && npm install && node server.js`
3. `cd frontend && npm install && npm run dev`
